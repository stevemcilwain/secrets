# Regex - Basics

```
^ start of a string
$ end of a string
. wildcard
| matches chars on either side (or)
\ escape
```

Special chars

```
\s whitespace
\S non-whitespace
\w word
\W non-word
\d one digit
\D one non-digit
\c control character
\n newline
\t tab
\r return
\0 null
```

Grouping

```
(xyz) group
(?:xyz) non group
[xyz] range
[^xyz] non range
[a-z] char in range
[0-9] digit in range
```
